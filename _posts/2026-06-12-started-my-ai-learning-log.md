Set up a minimalist GitHub Pages site to track what I learn on my AI journey.
With the Jekyll version, each learning is its own **Markdown file** in `_posts/`
with a little front-matter for the title, date, and tags.

No JSON to hand-edit — just write Markdown. GitHub builds the HTML on every push.
